# TEEKEH Brand Guidelines

> This document defines the visual identity, brand voice, typography, colors, and UI direction for the TEEKEH Inventory Tracker. It serves as the single source of truth for designers, developers, and AI agents to ensure consistency across the product.

---

# Brand Overview

## Brand Name

**TEEKEH**

## Industry

Retail • Fashion Accessories • Lifestyle

## Products

- Wristwatches
- Smart Watches
- Sunglasses
- Bags
- Wallets
- Belts
- Caps
- Cufflinks
- Perfumes
- Socks
- Ties & Clips
- Other Fashion Accessories

---

# Brand Personality

TEEKEH should communicate:

- Premium
- Modern
- Elegant
- Trustworthy
- Minimal
- Organized
- Professional

The experience should feel like walking into a clean luxury accessories store rather than a generic inventory application.

---

# Brand Mission

To simplify inventory management for TEEKEH while reflecting the premium quality of the products the business sells.

---

# Design Principles

The interface should be:

- Clean
- Spacious
- Easy to scan
- Fast to navigate
- Professional
- Data-focused

Avoid visual clutter.

Whitespace is part of the design.

---

Use for:

- Headings
- Dashboard titles
- Statistics
- Hero sections
- Cards
- Navigation titles

Recommended weights:

- SemiBold (600)
- Bold (700)
- ExtraBold (800)

Use for:

- Body text
- Tables
- Forms
- Labels
- Inputs
- Buttons
- Captions
- Reports

Recommended weights:

- Regular (400)
- Medium (500)
- SemiBold (600)

# UI Style

The application should feel inspired by:

- Stripe Dashboard
- Linear
- Notion
- Vercel Dashboard

Characteristics

- Flat Design
- Soft shadows
- Large spacing
- Rounded corners
- Strong typography
- Premium feeling

Avoid gradients unless approved.

Avoid glossy effects.

---

# Border Radius

Cards

```
16px
```

Buttons

```
12px
```

Inputs

```
10px
```

Modals

```
20px
```

---

# Shadows

Use soft elevation only.

Avoid heavy shadows.

Example

```
0 6px 24px rgba(0,0,0,.08)
```

# Imagery

Product photography should:

- Have clean backgrounds
- Be high resolution
- Focus on the product
- Maintain consistent aspect ratios

---

# Logo Usage

The logo should always have adequate whitespace.

Never stretch or distort it.

Prefer dark logo on light backgrounds.

Use white logo on dark navigation.

---

# Brand Voice

Tone

- Professional
- Friendly
- Clear
- Confident

Avoid overly technical language for business users.

Examples

Good

"Product added successfully."

"Stock updated."

"Low stock alert."

Bad

"Inventory mutation has completed successfully."

---

# Accessibility

Maintain WCAG AA contrast ratios.

Do not rely solely on color to communicate status.

All interactive elements should have clear focus states.

---

# Design System Rules

- Use an 8px spacing system.
- Prioritize consistency over creativity.
- Every component should be reusable.
- Minimize visual noise.
- Use typography hierarchy instead of excessive color.

---

# Future Brand Assets

As the brand evolves, add:

- Official logo files (SVG, PNG)
- Brand icon
- Favicon
- Product photography guidelines
- Social media assets
- Marketing templates

---

# Final Guiding Principle

The Inventory Tracker should feel like a premium business tool that reflects the quality of TEEKEH's products. Every design decision should reinforce simplicity, trust, and elegance while enabling users to manage inventory efficiently.
